# Retro Future Bulgaria

Retro Future Bulgaria is a cinematic, interactive timeline explorer that imagines alternative futures for Bulgarian cities across history, dystopia, nostalgia, and speculative techno-cultures.

## Features

- Curated timeline generator with 20+ handcrafted scenarios
- Filter by city, year, and scenario or generate a random timeline
- Save and manage favorites using LocalStorage
- Copy timeline descriptions to clipboard
- Dark, neon, glassmorphism UI with animated reveals
- Fully static, Netlify-ready frontend (no backend)

## Tech Stack

- Vite
- Vanilla JavaScript (ES modules)
- CSS (custom, no frameworks)
- LocalStorage

## Local Setup

Install dependencies:

- `npm install`

Start the dev server:

- `npm run dev`

Build for production:

- `npm run build`

Preview the production build:

- `npm run preview`
